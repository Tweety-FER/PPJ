#########################
## Upute za testiranje ##
#########################

I) Priprema:
  - Napraviti compile src fileova u direktorij bin/

II) Postavljanje test fileova:
  - Postaviti ppj1.sh i direktorij tests u bin/

III) Izvršavanje test fileova.
  - Pokrenuti bash skriptu (npr. bash ppj1.sh)

IV) Provjera pogrešaka
  - Pogledati direktorij "krivi", sadrži dani, te potom točan ispis

V) Čišćenje test fileova
  - Izbrisati bash skriptu, direktorij tests, direktorij krivi
